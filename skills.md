# Skills — Diseñados para amar

## Skills disponibles

| Skill | Archivo | Descripción |
|-------|---------|-------------|
| humanizer | `skills/humanizer/SKILL.md` | Detecta y elimina patrones de escritura IA. Hace el texto más natural y humano. |
| footnotes | `skills/footnotes.md` | Genera notas a pie de página con referencias, citas y ampliaciones sin sobrecargar el texto principal. |
| latex-template | `skills/latex-template.md` | Diseña y mantiene la plantilla LaTeX del libro (A5, dos caras, tipografía, márgenes). |
| doctrine-search | `skills/doctrine-search.md` | Busca documentos de doctrina católica, teología y antropología cristiana. |
| epub-build | `skills/epub-build.md` | Genera la versión EPUB del libro a partir de las fuentes LaTeX. |
| structure | `skills/structure.md` | Ayuda a diseñar la estructura narrativa de capítulos y secciones. |

## Uso

Invocar con `/nombre-skill` seguido del texto o instrucciones. Ejemplo:

```
/humanizer [texto a revisar]
/footnotes [párrafo donde insertar notas]
/doctrine-search Teología del Cuerpo sobre la complementariedad
/latex-template configurar capítulo nuevo
/epub-build compilar libro
/structure planificar capítulo sobre el amor conyugal
```
